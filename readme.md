这是一个基于Python + C++(dll)的自动套利程序  
控制基于TGBOT
其中图构建(三角套利等)和大量计算均使用C++完成  
单次循环在7600x测试机上可以做到6ms(最差情况50w个边的状态)  

首次使用需要更改主py文件的66-76行的所有API(XTapi可以不填,这是一个第三方交易所)
